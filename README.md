
# Stochastic Interest Rate Modelling and Yield Curve Prediction

This repository contains a complete Google Colab project for stochastic interest rate modelling using the Cox-Ingersoll-Ross (CIR) model. The project reconstructs longer-maturity zero-coupon yield curves from the 3-month yield, which is treated as a proxy for the instantaneous short rate.

The workflow follows the finance project problem statement: data preprocessing, base CIR implementation, model calibration, yield curve prediction, extension modelling, out-of-sample evaluation, and critical analysis.

## Project Objective

The main goal is to predict the yield curve for longer maturities using only the 3-month yield available during the test period. The model is trained on historical yield curve data containing maturities from 3 months to 30 years.

The project evaluates whether a one-factor short-rate model can reconstruct the curve accurately and then improves the base model with a dynamic CIR++ style extension.

## Files

- `CIR_Interest_Rate_Project_Colab.ipynb` - main Colab notebook containing the full project code and report.
- `base_cir_predictions.csv` - predictions generated by the base CIR model.

## Methodology

The notebook includes the following steps:

1. Data cleaning and preprocessing
2. Handling missing values, formatting issues, and outliers
3. Implementation of the CIR short-rate model
4. Calibration of CIR parameters: `kappa`, `theta`, and `sigma`
5. Reconstruction of the yield curve from the 3-month yield
6. Implementation of a dynamic CIR++ residual correction model
7. Out-of-sample evaluation using R2, RMSE, and MAE
8. Discussion of model assumptions, limitations, and practical interpretation

## Models Implemented

### Base CIR Model

The CIR model describes the short rate using a mean-reverting square-root diffusion process:

```text
dr_t = kappa * (theta - r_t) dt + sigma * sqrt(r_t) dW_t
```

The model uses the closed-form CIR zero-coupon bond pricing formula to reconstruct yields at different maturities.

### Dynamic CIR++ Extension

The base CIR model is extended with a maturity-specific residual correction:

```text
y_ext(t, tau) = y_CIR(t, tau) + phi_tau(x_t)
```

Here, `x_t` is built only from the 3-month yield series. This helps capture slope and curvature effects that a single-factor CIR model cannot fully explain.

## Results

On the provided dataset, the notebook achieved:

- Base CIR Overall R2: approximately `0.893`
- Dynamic CIR++ Overall R2: approximately `0.915`

The extended model clears the required out-of-sample R2 threshold of `0.85`.

## Key Takeaway

A one-factor CIR model provides a strong structural baseline for short-end yield curve reconstruction, but it struggles to capture all slope and curvature movements. The dynamic CIR++ extension improves out-of-sample performance while respecting the constraint that only the 3-month test yield is used for prediction.
